# Repeated Measurements Analysis — Exam Report

## Overview
This repository contains the R Markdown analysis for the repeated measurements exam.
The report covers descriptive statistics and marginal model analyses of clustered 
longitudinal data, with a focus on aortic gradient and aortic regurgitation outcomes 
following cardiac surgery.

## Repository Structure
Each branch corresponds to a question in the exam:

| Branch | Content |
|---|---|
| `main` | Data loading, cleaning, and shared setup |
| `question-1` | Question 1 analysis |
| `question-2` | Question 2 analysis |
| `question-3` | Question 3 analysis |
| `question-4` | Question 4 analysis |

## Report Structure
Each question is split into two sections:
- **Descriptive Statistics** — exploratory plots and summary statistics
- **Main Analysis** — model fitting, selection, and interpretation
